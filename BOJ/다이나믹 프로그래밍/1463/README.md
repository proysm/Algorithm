1. 테이블 정의하기
- D[i]=i를 1로 만들기 위해 필요한 연산 사용 횟수의 최솟값
2. 점화식 찾기
- 예를 들어, D[1], D[2], ..., D[11] 의 값을 다 안다고 가정하고 D[12]를 생각해본다.
- 가능한 연산은 3가지 이다.
    - 3으로 나누기 D[12] = D[4] + 1
    - 2로 나누기 D[12] = D[6] + 1
    - 1 빼기 D[12] = D[11] + 1

- 따라서 점화식은 다음과 같다.
    - 3으로 나누기 D[k] = D[k/3] + 1
    - 2로 나누기 D[k] = D[k/2] + 1
    - 1 빼기 D[k] = D[k-1] + 1
