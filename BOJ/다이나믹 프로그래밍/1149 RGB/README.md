# 2579 계단 오르기

1. 테이블 정의하기
- a[i][k] : i번째 집을 k색깔로 칠했을 때 최소값
- 이때, 가격을 이 배열에 둔다. house[i][k] : i번째 집을 빨강(k=1), 초록(k=2), 파랑(k=3)으로 칠할 때 가격

2. 점화식 찾기
- a[i][1] = min(a[i-1][2], a[i-1][3]) + house[i][1];
- a[i][2] = min(a[i-1][1], a[i-1][3]) + house[i][2];
- a[i][3] = min(a[i-1][1], a[i-1][2]) + house[i][3];

3. 초기값 정의하기
- a[1][1] = house[1][1];
- a[1][2] = house[1][2];
- a[1][3] = house[1][3];
