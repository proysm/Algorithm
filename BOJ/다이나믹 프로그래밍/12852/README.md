# 12852 1로 만들기 2

### 문제의 핵심
기존 1463(1로 만들기) 문제에서 경로를 돌아가야 한다.
경로를 저장하는 배열을 잘 활용해야 한다.

1. 테이블 정의하기
- d[i] : 정수 i를 세 가지 연산을 이용하여 1을 만드는 횟수의 최소값
- pre[i] : 정수 i를 1을 만드는 경로 중에 그 전 경로의 index를 저장함

2. 점화식 찾기
- 이 중 가장 작은 값이 d[i]에 들어갈 것이다.
    - d[i] = d[i-1] + 1;
    - d[i] = d[i/2]+1;
    - d[i] = d[i/3]+1; 

3. 초기값 정의하기
- d[1] = 0;
- pre[1] = 1;
