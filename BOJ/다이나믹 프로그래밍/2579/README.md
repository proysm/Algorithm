# 2579 계단 오르기

1. 테이블 정의하기
- D[i][j] 현재까지 k개의 계단을 밟았고 i번째 계단을 밟았을 때, 점수의 최대값

2. 점화식 찾기
- k가 1 또는 2 밖에 될 수 없다. 왜냐하면 연속된 3개의 계단을 밟지 못한다고 했다.
- 따라서 다음과 같은 점화식이 나온다.
    - d[i][1] = max(d[i-2][1], d[i-2][2]) + score[i];
    - d[i][2] = d[i-1][1] + score[i];

3. 초기값 정의하기
- d[1][1] = score[1];
- d[2][1] = score[2];    
- d[2][2] = score[1] + score[2];
