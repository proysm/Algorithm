### 문제 풀이

#### ⭐️ 문제 핵심은 4진법을 이용하는 것이다. ⭐️

```C++
for(int tmp=0; tmp<(1<<(2*cctv.size())); tmp++)
```
한 cctv 마다 네가지 방향으로 회전이 가능하다. (동서남북)

따라서, cctv가 3개라고 한다면 총 64방향 회전의 경우의 수가 존재한다. (4*4*4)

```C++
for(int i=0; i<cctv.size(); i++)
```
cctv를 하나씩 본다.

```C++
int dir = brute % 4; // 방향을 결정
brute /= 4; // 다음 방향을 결정
```
해당 cctv의 방향을 결정한다.

4진법을 사용했기 때문에 4로 나눈 나머지를 이용하고, brute/4를 하는 이유는 나누기로 4진법의 자리수를 따라가는 것을 생각하면 될 것 같다. (나는 이해가 잘 안돼서 직접 해봤다)

**Example**

    54 % 4 = 2, 54 / 4 = 12

    12 % 4 = 0, 12 / 4 = 3

    3  % 4 = 3, 3  / 4 = 3
